## Description 📕

This application was built using Node.js and Express.js for the back end, and can be used to write and save notes. The data from the notes is currently saved and read from a JSON file.note

### Installation 🔍

```
git clone
cd my-note-taker
npm i
```

### Launch the app 🚀

```
npm run start
```
